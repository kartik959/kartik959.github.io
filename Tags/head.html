The <head> element is a container for all the head elements. The <head> element must contain the title of the document (title), which can contain scripts, styles, meta information, and more information.
The elements listed below can be used inside the element:

  • <title>
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>
