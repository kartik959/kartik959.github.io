HTML Arrtibute

The attribute provides additional information for the HTML element.

HTML tags can have attributes. The attribute provides more information about the HTML element.
Properties always appear in the form of name / value pairs, such as name= "value".
Properties are always specified in the start tag of the HTML element.

Example

The HTML link is defined by the <a>label. The address of the link is specified in the href attribute:

<a href="url">This is a link </a>
