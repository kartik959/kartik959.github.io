TIn HTML, the image is defined by the <img> label.
<img> is an empty label, meaning that it contains only attributes, and there is no closed label.
To display the image on the page, you need to use the source property (SRC). SRC refers to "source". The value of the source attribute is the URL address of the image. The syntax for defining images is:<img src="url" />

You can use the <img> tag to put the picture on your web page and you can set the size of the picture by setting the width and the height property of the picture.

Example

<img src ="4447H.jpg" width="200px" height="200px" >
