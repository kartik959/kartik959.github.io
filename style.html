HTML Style

The style attribute is used to change the style of the HTML element.

A general method of changing the style of all HTML elements is provided.
The style attribute provides many ways to change your web page style

Example

<h2 style="background-color:red">This is a heading</h2>

<p style="background-color:green">This is a paragraph.</p>
