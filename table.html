The table is defined by the <table> label. Each table has a number of rows (defined by the <tr> label), and each row is divided into a number of cells (defined by the &td& label). The letter TD refers to the table data (table data), that is, the content of the data cell. A data cell can contain text, pictures, lists, paragraphs, forms, horizontal lines, tables, and so on.

Example:

<table border="1">
<tr>
<td>row 1, cell 1</td>
<td>row 1, cell 2</td>
</tr>
<tr>
<td>row 2, cell 1</td>
<td>row 2, cell 2</td>
</tr>
</table>

row 1, cell 1 row 1, cell 2
row 2, cell 1 row 2, cell 2
